# EduAssist — AI-помощник для школы

### Хакатон «Алиса AI» | 2026

---

## Проблема

Учителя тратят часы на рутину — проверку работ, составление расписания, анализ дисциплины.  
Директора не имеют объективных данных о качестве уроков.  
Ученики получают одинаковый подход вне зависимости от своих пробелов.

> **Результат:** выгорание учителей, слепые зоны у руководства, отстающие дети.

---

## Решение

**EduAssist** — платформа с тремя рабочими пространствами (директор / учитель / ученик),  
где Алиса AI является центральным инструментом обработки данных, а не просто чатом.

---

## Как это работает

### Для ученика
1. Загружает фото или PDF домашней работы
2. Алиса (OCR + YandexGPT) анализирует работу:
   - выставляет оценку с обоснованием
   - указывает конкретные ошибки
   - задаёт 3 наводящих вопроса для самопроверки
3. Слабые темы фиксируются в персональном профиле

> Алиса не даёт готовый ответ — она учит думать.

---

### Для учителя
1. Нажимает «Начать урок» — система слушает класс в фоне (Yandex SpeechKit)
2. Алиса фильтрует фоновый шум, реагирует только на явные обращения («Алиса, ...»)
3. При повышении уровня шума — мягкое уведомление учителю
4. По окончании урока: тезисная сводка + график активности класса
5. Все домашки учеников проверены автоматически — учитель видит сводный отчёт

---

### Для директора
1. Загружает Excel с учителями, классами и предметами
2. Алиса анализирует ограничения и генерирует расписание:
   - без конфликтов (два предмета в одно время)
   - без неудобных «окон» у учителей
   - с учётом предпочтений педагогов
3. Дашборд метрик: уровень шума по классам, качество домашних работ, прогресс учеников

---

## Технологии Алиса AI / Yandex AI Studio

| Возможность | Применение |
|------------|-----------|
| **YandexGPT** | Проверка ДЗ, генерация вопросов, расписание, наставнический диалог |
| **SpeechKit STT** | Реал-тайм анализ урока, распознавание речи |
| **Yandex Vision OCR** | Перевод фото тетрадей в текст |
| **Function Calling** | `checkHomework()`, `generateSchedule()`, `analyzeLesson()` |
| **Контекст / память** | Профиль ученика передаётся как системный контекст в каждый запрос |
| **Агентный сценарий** | OCR → анализ → генерация вопросов → сохранение в профиль |
| **Мультимодальность** | Текст + изображения + аудио одновременно |

## Что реализовано

- Три роли с разграниченным доступом (JWT-авторизация)
- Загрузка и хранение файлов (MinIO)
- Проверка домашних работ: OCR + анализ + персонализированный отчёт
- Реал-тайм мониторинг урока с анализом шума
- Генерация расписания из Excel с проверкой ограничений
- Персональный AI-профиль ученика с историей работ и слабыми темами
- Дашборд директора с графиками и метриками

---

## Ограничения (честно)

- Рукописный текст: OCR работает хуже, чем с печатным — для MVP допустимо
- Фильтрация голоса учителя: в MVP — по уровню шума, точная диаризация — следующий этап
- Расписание: GPT генерирует первый вариант, директор может корректировать вручную

---

## Потенциал развития

- Интеграция с дневниками (электронный журнал, Госуслуги)
- Мобильное приложение с голосовым интерфейсом для учеников
- Поддержка видеоаналитики (зрительный контакт, вовлечённость)
- Рекомендации учебных материалов на основе профиля слабых тем
- B2B: лицензирование школам, районным управлениям образования
